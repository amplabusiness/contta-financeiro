# DIAGNÓSTICO TÉCNICO — AMPLA FINANCEIRO
Data: 28/01/2026

## 1. Visão Geral
O sistema Ampla Financeiro já possui estrutura avançada, porém apresenta problemas de governança de dados.

## 2. Principais Problemas
- Saldo de abertura não propagado
- Contas a receber fora do plano
- Entries órfãos
- Múltiplas fontes de verdade

## 3. Conclusão
Necessário núcleo contábil único e transacional.
